socialShare plugin for Dotclear 2
=================================

Summary
-------

This plugin adds social share buttons to posts and/or pages, for Twitter, Facebook, Google+ social networks and by mail.

Default style for social share links
------------------------------------

    margin-top: 1.5em;
    margin-bottom: 1.5em;
